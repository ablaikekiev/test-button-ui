# Getting Started

## Библиотека доступна по команде
### `npm i ablai-button-ui`

## Available Scripts

### `npm install` 
Для установки зависимостей

### `npm run publish` 
Билдит сборку и публикует библиотеку под новой версией

### `npm run storybook` 
Билдит storybook для просмотра компоненты
